About BloodBridge

A cloud-native blood donation management system built with modern web technologies and AWS architecture.

Our Mission

BloodBridge is designed to bridge the gap between blood donors, hospitals, and blood banks through innovative cloud technology. Our platform ensures that lifesaving blood resources are efficiently managed and distributed.

Built as a demonstration of modern full-stack development with cloud integration, BloodBridge showcases how healthcare applications can leverage AWS infrastructure for reliability, security, and scalability.

Every feature is designed with the user in mind - from the intuitive donor dashboard to the powerful admin analytics, ensuring that saving lives is just a few clicks away.

Local Development Setup

Run BloodBridge locally for development and demonstration

Prerequisites

Python 3.8+
MySQL 8.0 (XAMPP/WAMP)
pip package manager
Git

Installation

git clone repo
cd BloodBridge
pip install -r requirements.txt
python app.py

Database Setup

CREATE DATABASE bloodbridge_db;
USE bloodbridge_db;
SOURCE database/schema.sql;

AWS Deployment

1. Launch EC2 instance
2. Install Python & MySQL client
3. Configure RDS endpoint
4. Deploy with Gunicorn
5. Setup ALB + Route 53
