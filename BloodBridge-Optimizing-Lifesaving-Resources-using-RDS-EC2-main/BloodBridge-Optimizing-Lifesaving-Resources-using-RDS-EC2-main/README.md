## BloodBridge: Optimizing Lifesaving Resources using RDS & EC2

BloodBridge is a smart blood donation and management platform developed to connect donors, hospitals, blood banks, and administrators through a centralized digital system. The project aims to improve emergency blood availability, reduce response time, and simplify blood inventory management using modern web technologies and cloud architecture concepts.

The system provides separate modules for Admin, Donor, Hospital, and Blood Bank Manager. Donors can register and manage their profiles, hospitals can raise emergency blood requests, blood banks can maintain blood stock records, and admins can monitor all system activities through a dashboard.

BloodBridge is developed using Python Flask, MySQL, HTML, CSS, JavaScript, and Bootstrap. The project also includes AWS cloud architecture references such as Amazon EC2, Amazon RDS, Load Balancer, CloudFront, and VPC for scalable and secure deployment concepts.
